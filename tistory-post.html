🃏 짝을 찾아라! - 메모리 카드 게임

카드를 뒤집어 같은 그림의 짝을 맞춰보세요!

* 모바일에서도 플레이 가능합니다

🎮 게임 소개

카드를 2장씩 뒤집어서 같은 그림의 짝을 찾는 클래식 메모리 게임입니다.
집중력과 기억력을 키울 수 있어 남녀노소 누구나 즐길 수 있어요!

🕹️ 게임 모드

🎯 솔로 모드 혼자서 최소 시도로 모든 짝을 맞춰보세요
🏆 스테이지 모드 쉬움 → 보통 → 어려움, 3단계를 연속으로 클리어!
⚔️ 2인 대전 친구와 번갈아가며 짝 맞추기 대결
⏰ 타임어택 제한 시간 안에 모든 짝을 맞춰야 해요
📅 오늘의 도전 매일 바뀌는 테마와 난이도로 도전!

✨ 주요 기능

  • 40가지 테마 - 동물, 과일, 포켓몬, 공룡, 우주 등 다양한 카드 테마
  • 3단계 난이도 - 쉬움(4x3), 보통(4x4), 어려움(5x4)
  • 테마 해금 시스템 - 게임을 클리어할 때마다 새로운 테마가 열려요
  • 콤보 시스템 - 연속으로 짝을 맞추면 콤보 보너스!
  • 힌트 기능 - 막힐 때 힌트 버튼으로 도움 받기
  • 랭킹 & 통계 - 내 기록을 저장하고 순위를 확인
  • 업적 시스템 - 다양한 조건의 업적을 달성해보세요
  • 다크 모드 - 눈이 편한 다크 모드 지원

📊 난이도 안내

💡 플레이 팁

  1. 처음 몇 장은 일부러 다양한 위치를 뒤집어 카드 위치를 기억하세요
  2. 연속으로 짝을 맞추면 콤보가 올라가요!
  3. 막힐 때는 힌트 버튼을 활용하세요 (횟수 제한 있음)
  4. 게임을 클리어하면 새로운 테마가 해금됩니다

React + TypeScript + Tailwind CSS로 만든 웹 게임입니다 🎨
